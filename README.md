# Simple Interest Calculator

## Description

This project demonstrates how to calculate simple interest using a basic mathematical formula.

## Formula

Simple Interest (SI) = (P × R × T) / 100

Where:

* P = Principal amount
* R = Rate of interest
* T = Time (in years)

## Example

If:
P = 1000
R = 5%
T = 2 years

Then:
SI = (1000 × 5 × 2) / 100 = 100

## Purpose

This project is created as a simple demonstration of using GitHub and documenting a basic calculation.
